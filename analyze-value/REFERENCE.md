## 1. 估值基准线

财富是一个人能调动资源的总量，是**相对值**，因此需要一个统一的对比基准。这个基准是**风险尽可能小的资产**——由国家背书的现金存款：

- 中国：各大银行人民币定期存款利率
- 美国：FDIC 担保（25 万美元以下）的银行存款 / 短期国债利率

**任何资产的价值，都不是孤立的，而是相对于「把钱存进银行」这个安全选项来衡量的。** 整套估值方法的折现率 `r`，就来自这条基准线。

银行存款被视为安全：规范银行的储蓄在过去几十年基本没有取不出来的情况，因此把现金储蓄当作理想基准线。

## 2. 期望收益：给风险固收资产估值

对于票据、债券、理财等带违约风险的固定收益资产，**先算期望收益，再和存款基准线比较**。

**步骤一：算期望收益。** 把各种到期结果按概率加权：

> 例：面值 1 万元、承诺一年后付 10500 元（+5%）的商票。假设：
> - 87% 概率本息全拿到 → 10500
> - 5% 概率只收回本金 → 10000
> - 5% 概率只收回一半本金 → 5000
> - 3% 概率本息全损 → 0
>
> 期望值 E = 10500×87% + 10000×5% + 5000×5% + 0×3% = **9885 元**

**步骤二：与基准线比较（折回今天）。** 9885 元是一年后的期望值，折回今天：
9885 / (1+3%) = **9597 元**。也就是说，存 9597 元进银行、3% 利率，一年后同样能拿到 9885 元。

**结论：纸面 1 万元的商票，真实价值只相当于 9597 元。**

> 实务提醒：很多「比储蓄高一两个百分点」的理财产品，本质是公司债券/商票的变体或打包。交进去 1 万元，真实价值可能只剩 9000 多元。

脚本用法：
```bash
python3 scripts/dcf_valuation.py --expected 10500:0.87 10000:0.05 5000:0.05 0:0.03 --r 0.03
```

## 3. 自由现金流与估值公式（DCF）

经典企业估值公式：

```
            n     FCFₜ
Value  =   Σ   ─────────
           t=1   (1 + r)ᵗ
```

- **Value**：企业创造的价值（直接决定估值）。
- **FCF（自由现金流，Free Cash Flow）**：一般指主营业务的自由现金流。
- **r**：无风险回报率，通常取银行储蓄利率。
- **t**：第几年；**n**：年限，考虑长期/终身回报，通常 n ≥ 10 年。

一句话：**企业价值 = 未来每一期自由现金流的现值之和。**

### 为什么用自由现金流，不用净利润？

- 净利润**容易造假**；
- 很多企业的净利润将来都要**再投入再生产**，并不是真正能拿到手的钱。
- 自由现金流是「真金白银」，更能反映企业的造血能力。

### 为什么公式里没有「成本」？

公式变量里**没有成本一项**。这是现代投资理念的精髓：

- 一个经营型资产（企业）的估值，和它当初花多少钱建造、包含多少固定资产**没有太大关系**。
- 把当初造价、固定资产按市价单独拿出来算，是**静态思维**。现代企业的固定资产价值相对企业价值通常很低。
- 投资时盯着固定资产而非盈利能力，回报率通常不高——前者获利一次性，后者是长期的。

**所以看一家企业估值，主要看它当下和未来能持续产生多少现金流。**

### 为什么要折现（打折扣）？

5 年后的 1 万元 ≠ 今天的 1 万元。今天拿到，就有 5 年时间让它继续赚钱。因此每年现金流都要按折扣因子（利率 r）折回今天。这一步即「折扣现金流 DCF」。

> 仅用 r 折现的算例：A 公司第一年 t=1，DCF = 500/(1+3%)¹ = 485.4 万；第二年 t=2，DCF = 1020/(1+3%)² = 961.4 万。

## 4. 风险对估值的影响

只用银行利率 r 折现还不够。**有风险的资产，折扣应该比无风险利率更大。** 不能用无风险投资的收益去对标带风险的投资——这是理财最常犯的错误之一。

做法：把市场风险 **β** 纳入折现因子（CAPM 思路）。

- 把整个市场的风险看作 **1**；单只股票通常 **β > 1**（波动比大盘大）。
- 业务越不稳、风险越高 → β 越大 → 折扣越大 → 估值越低。

风险调整后的折现因子：

```
d = 1 + r + β·(rm − r)
```

其中 `rm` 为市场普遍回报（原文取 5%）。完整估值公式：

```
            n            FCFₜ
Value  =   Σ   ──────────────────────
           t=1   [1 + r + β·(rm − r)]ᵗ
```

> 算例：r=3%，rm=5%，A 公司 β=2 → d = 1+3%+2×(5%−3%) = 1.07。
> 第一年 DCF = 500/1.07¹ = 467.3 万；第二年 = 1020/1.07² = 890.9 万。
> （对比仅用 r 折现的 485.4 / 961.4，风险折扣后明显更低。）

**同样的现金流，业务越不稳、风险越高，估值打的折扣越大，企业实际价值越低。**

## 5. 从折现现金流到估值的两种口径

把逐年 DCF 求和得到 `ΣDCF` 后，有两种把它表述为「估值」的口径，报告时务必说明用的是哪种：

### 口径 A — 标准现值口径（金融教科书）
`ΣDCF` 本身就是未来现金流折算到今天的现值，即估值 = ΣDCF。

### 口径 B — 吴军「等价存款本金」口径（原文采用）
问：需要多少本金 P 存银行、n 年、按无风险利率 r 复利，才能产生与 ΣDCF 等额的利润？

```
P · [(1 + r)ⁿ − 1] = ΣDCF      →      估值 = ΣDCF / [(1 + r)ⁿ − 1]
```

> 原文算例（不含风险折扣时）：A、B 两公司 10 年折现现金流分别为 6478 万、8703 万；
> 对应估值 = 6478 / (1.03¹⁰−1) ≈ **18835 万**，8703 / (1.03¹⁰−1) ≈ **25305 万**。
> 含风险折扣后，折现现金流降为约 5227 万、6708 万，估值也相应大幅下降（原文「大致」给到约 1.74 亿、2.33 亿，含取整）。

两种口径的差别只是表述方式；脚本会同时打印两者。投资决策更常用口径 A。

## 6. 商业地产的估值

能产生现金流的资产都可借鉴这套方法，**商业地产**（商铺/写字楼/收租物业）即是。

- 估法与企业相同：估算未来逐年自由现金流 → 用 `d = 1 + r + β·(rm − r)` 折现求和 → 换算估值。
- **自住房不能这么算**，它更多看供需关系。
- 商业地产**不用单独看供需**：供求关系已经体现在自由现金流里（求大于供 → 租金上涨 → FCF 提高，反之下降），再算供需就重复了。

> 原文算例：某商业地产 10 年前售价 1 亿。设 β=1.5、rm=5%、r=3% → d=1.06。
> 累计折现自由现金流 ≈ 2693 万；估值 = 2693 / (1.03¹⁰−1) ≈ **7830 万**。
> 现值已不如 10 年前 1 亿的售价。再考虑维护成本、房屋折旧，商业地产长期并非高回报投资。

## 7. 原文完整算例

### 公司 A vs 公司 B（节选）
两家公司过去 10 年自由现金流不同：A 业绩不稳、时好时坏；B 起步艰难但持续发展。银行利息 3%。

- 不考虑风险（仅按 3% 折现）：10 年折现现金流 A=6478 万、B=8703 万 → B 价值更大。
- 若直接对标存款：需存 18835 万、25305 万 @3% 存 10 年才能产生等额利润。
- **但不能用无风险收益对标风险投资**：买下 A 可能几年后持续亏损。
- 引入风险：A β=2、B β=1.5，r=3%、rm=5% → 10 年折现现金流降为 5227 万、6708 万；估值大致降到约 1.74 亿、2.33 亿。

> 注：原文只给出了部分年度 FCF 明细（首两年 500、1020）。脚本可对任意完整 FCF 序列复现逐年折现，逐年数字与原文一致（467.3 / 890.9）。

### 思考题（原文）
回顾你持有或关注的某家公司，它的自由现金流稳定吗？

## 8. 实操检查清单

给任意企业/资产估值时，依次回答：

1. **基准线**：无风险利率 r 取多少？（中国定存 / 美国国债）
2. **现金流从哪来**：主营业务 FCF 的来源、商业模式是否清晰？
3. **稳不稳**：未来 n 年 FCF 可持续吗？增长/平稳/衰退？周期性强不强？
4. **风险多大**：β 该取多少？业务越不稳 β 越高。
5. **算估值**：用脚本跑 DCF，给出区间（对 β、增长率做高/中/低敏感性分析）。
6. **对比**：估值 vs 当前市价 vs 历史成本——高估还是低估？主要驱动假设是什么？
7. **方向性正确**：不追求精确到个位，追求判断方向对、假设站得住。

## 9. 常见误区

| 误区 | 正确做法 |
|------|----------|
| 按当初造价 / 固定资产市值估值 | 撇开成本，只看未来现金流 |
| 用净利润衡量盈利能力 | 用自由现金流（净利润易造假、要再投入） |
| 用无风险收益对标风险资产 | 引入 β，风险越高折扣越大 |
| 盯着历史股价判断「便宜/贵」 | 历史价格不进公式，看内在现金流价值 |
| 理财「比存款高一两个点」就买 | 先算期望收益、折回今天再比基准线 |
| 给商业地产再单独叠加供需溢价 | 供需已含在 FCF 里，不重复计 |
| 用现金流法给自住房估值 | 自住房看供需，不用此法 |
| 追求一个精确的估值数字 | 给区间、做敏感性分析，方向性正确即可 |
